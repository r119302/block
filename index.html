ゲーム紹介

マウスのカーソルとボールを弾くプレートの右端が連動して動きます。プレイ画面の外にマウスカーソルが出てしまうと操作できなくなるので要注意。 ブロックが１つ消えるたびにscoreが１０増え、全てのブロックが消えれば、またすぐに復活します。つまり、ボールを落とさない限り永遠にプレイすることができるということです。 GAME OVERになっても大丈夫。ワンクリックでスタート画面に戻れます。さあ、よい暇つぶしを！！

こだわり

こだわったのは「何回でも繰り返し遊べるところ」、「難しい要素もなく、ただひたすらにブロックを消すだけというシンプルさ」。 ブロックの復活は全てのブロックが消えて、プレートにボールが当たった後すぐに復活できるようにしました。また、START, GAME PLAY, GAME OVERの画面の移り変わりもスムーズに行くように作りました。 難しかったのはやはり当たり判定でブロックの角や辺に当たるときの判定が特に難しかったです。
